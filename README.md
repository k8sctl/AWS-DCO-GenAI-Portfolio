# AWS DCO GenAI Portfolio

> AWS DCO(Data Center Operations) 인턴십 직무 이해를 위한 학습형 포트폴리오

---

## 1. 과정명

**AWS DCO 인턴십 직무 이해를 위한 생성형 AI 활용 및 포트폴리오 제작 과정**

## 2. 포트폴리오 목적

이 포트폴리오는 DCO 직무의 핵심 개념(SOP, Ticket, Escalation, Handover 등)을 이해하고, 생성형 AI를 활용해 로그 분석·문서 정리·자동화 스크립트 작성을 연습한 과정을 기록하기 위해 만들었습니다.

목표는 세 가지입니다.

1. DCO 직무 언어와 업무 흐름(이벤트 확인 → 기록 → 전달)을 이해한다.
2. 생성형 AI를 업무 보조 도구로 활용하되, 결과를 사람이 검토·수정하는 습관을 기른다.
3. 수업 산출물을 체계적으로 정리하여 학습 과정을 증명할 수 있는 포트폴리오를 완성한다.

## 3. 교육용 샘플 데이터 사용 안내

⚠️ **이 저장소의 모든 자료는 교육용 샘플 데이터로 작성되었습니다.**

- 실제 AWS 내부 SOP, 내부 문서, 내부 절차가 아닙니다.
- 모든 로그는 학습용으로 만들어진 샘플 로그입니다.
- 장비명은 `SAMPLE`, `DEMO`, `EDU` 접두어가 붙은 가상의 이름만 사용합니다.
- 이 포트폴리오의 내용은 실제 운영 환경에 적용하기 위한 것이 아니라, 직무 이해를 위한 학습 기록입니다.

```

├─ 01_dco_glossary/           # DCO 직무 용어집 (SOP, Ticket, Escalation, Handover 등)
├─ 02_dco_profile/            # DCO 직무 프로필 정리 (역할, 필요 역량, 업무 흐름)
├─ 03_sop_analysis/           # 교육용 샘플 SOP 분석 인증본 및 체크리스트
├─ 04_environment_setup/      # 실습 환경 구축 기록 (도구 설치, 설정 과정)
├─ 05_antigravity_cli/        # Antigravity CLI 학습 및 실습 기록
├─ 06_cli_file_automation/    # CLI 기반 파일 정리·자동화 실습
├─ 07_log_analysis_script/    # 샘플 로그 분석 스크립트 (Link Down 이벤트 추출 등)
├─ 08_incident_analysis/      # 샘플 로그 기반 인시던트 분석 연습
├─ 09_incident_report/        # Incident Report 및 Handover Memo 초안 작성 연습
├─ 10_stitch_dco_dashboard/   # Stitch를 활용한 DCO 대시보드 UI 설계
└─ 11_exam_prep/              # 지필시험 대비 요약 정리
```

## 5. 보안상 포함하지 않는 정보

이 포트폴리오에는 다음 정보를 **절대 포함하지 않습니다.**

- 실제 AWS 내부 SOP, 내부 문서, 내부 절차
- 실제 장비명, 장비 시리얼 번호
- 실제 IP 주소, 네트워크 구성 정보
- 실제 티켓 번호 및 티켓 내용
- 실제 고객명, 실제 회사 내부 정보
- 실제 운영 환경 접속 기록 또는 진단 명령 실행 결과
- 개인정보 (본인 및 타인의 연락처, 계정 정보 등)

또한 AI가 생성한 답변은 그대로 사용하지 않고, 원인을 단정하는 표현이나 부정확한 내용이 없는지 검토·수정한 뒤 반영합니다.
